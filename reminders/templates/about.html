What is PyToDo?
PyToDo is an interactive web app that helps users keep track of their list of to-do's as well as the corresponding deadlines. The app also randomly recommends a to-do to finish!
What is the class definition?
We created our own Reminder class that has fields such as reminderID (unique identifier), creator, body, and timestamps for creation and duedate. For the class, we used two magic methods: __repr__ and __setattr__ to account for setting the attributes as well as representing the class as a string for debugging.
What are the packages we used?
For first class packages, we used datetime for easy calculations of time as well as random for randomly generating tasks to complete. For third class packages, we used Django for the web framework, Djongo for storage on MongoDB, and humanfriendly for human readable format of datetime.
